**Card** — the standard surface container: hairline border, 8px corner, soft level-1 shadow. Use to group content on the gray canvas.

```jsx
<Card>
  <strong>Senior Engineer</strong>
  <p className="muted">Northwind · Remote</p>
</Card>
<Card interactive onClick={open}>…</Card>
```

Restrained by design: no colored left-border stripes, no gradients.
