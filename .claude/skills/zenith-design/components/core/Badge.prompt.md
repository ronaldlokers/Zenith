**Badge** — small uppercase pill for counts, flags, and meta chrome. Not for pipeline stage (use `StatusBadge`).

```jsx
<Badge>12 open</Badge>
<Badge variant="accent">New</Badge>
<Badge variant="warn">Gone quiet</Badge>
```

Variants: `default`, `accent` (soft teal), `warn` (gone-quiet heat). Full-radius, chrome-size uppercase type.
