**Icon** — Zenith's line-art glyph. currentColor stroke at weight 2; icons tint with surrounding text color (accent, muted, ink). No emoji or icon fonts in this brand.

```jsx
<Icon name="pipeline" />
<Icon name="bell" size={16} style={{ color: "var(--accent)" }} />
```

Names: overview, pipeline, feed, calendar, network, cv, settings, search, filter, archive, bell, error. `ICON_NAMES` exports the full list.
